# Run-Goals Slop Cleaner (Fragment)

**읽기 전용 탐지기.** 코드 편집, 파일 작성, 워크플로우 호출 금지.

## 검사 범위
현재 goal에서 변경된 파일만 검사. 변경 없으면 PASS 즉시 반환.

## 분류 체계

| 카테고리 | 설명 | 등급 |
|---------|------|------|
| **Fallback 코드** | 에러 마스킹, 묵살하는 catch, 근거 없는 기본값 | 차단 |
| **중복** | 복사-붙여넣기 로직, 반복 패턴 | 차단 |
| **데드 코드** | 미사용 변수/함수, 도달 불가 분기 | 차단 |
| **불필요한 추상화** | 단순 위임 래퍼, 투기적 인터페이스 | 권고 |
| **경계 위반** | 계층 역참조, 숨겨진 결합 | 차단 |
| **UI/디자인** | 12px 이하 폰트, 과도한 그림자, 하드코딩 색상 | 권고 |
| **테스트 누락** | 핵심 동작 미검증, 약한 회귀 커버리지 | 차단 |

## 보고 형식

```
AI SLOP CLEANUP REPORT
======================
범위: <변경 파일 목록>
모드: read-only scan

[차단 발견사항]
- <파일:라인> <카테고리> — <설명>

[권고 발견사항]
- <파일:라인> <카테고리> — <설명>

검사 파일: <N>개
결과: PASS | BLOCKED (<차단 개수>건)
```

## 완료 후 반환
구조화된 보고서를 run-goals 리더에게 반환. 직접 수정하지 않음.
차단 0건이면 `결과: PASS` 반환 후 종료.
